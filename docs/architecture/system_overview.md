# ClubOS System Overview

## Goal

Create a recurring monthly operating system for digital business teams.

## Main Layers

1. Source data
2. Databricks medallion pipeline
3. Gold outputs
4. Backend API
5. Frontend SaaS product
6. Optional AI explanation layer

## Core Product Flow

1. monthly files are uploaded
2. pipeline validates and standardizes them
3. KPI, benchmark, signal, and priority tables are rebuilt
4. backend exposes those outputs
5. frontend shows the same recurring workflow

## Product Hero

The hero feature is the Priority Board, powered by deterministic scoring over Gold outputs.
