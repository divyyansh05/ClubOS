Path A — Databricks Data Pipeline
Admin uploads monthly CSV/Excel files to Databricks DBFS /FileStore/real_madrid/monthly_data/YYYY-MM/
5 files (4 assets + 1 benchmark)
BRONZE Raw Ingestion — zero transformations, preserve source exactly
Pipeline · Bronze
bronze/01_ingest_internal_metrics.py
Reads 4 asset CSVs, adds ingestion_timestamp + source_file metadata → writes bronze.internal_metrics
Pipeline · Bronze
bronze/02_ingest_benchmark_metrics.py
Same for peer benchmark CSV (5 clubs × 8 metrics) → writes bronze.benchmark_metrics
SILVER Normalization — lowercase, null handling, outlier flags, dedup
Pipeline · Silver
silver/01_normalize_internal_metrics.py
Trim/lowercase metric_name, flag outliers (3σ), dedup by (month, asset, metric) → silver.internal_metrics
Pipeline · Silver
silver/02_normalize_benchmark_metrics.py
Same for benchmark. Dedup by (month, club_id, asset, metric) → silver.benchmark_metrics
Infra · Quality
silver/03_build_data_quality_checks.py
Schema, row-count, duplicate, date-gap, value-range checks → silver.data_quality_checks
GOLD Business Analytics — 5 final Delta tables, consumed by backend API
Pipeline · Gold
gold/01_build_kpi_health.py
6m trend slope, 12m volatility, persistence months, seasonal deviation → health_status (good/review/stable)
Pipeline · Gold
gold/02_build_peer_benchmark.py
Joins RM vs 5 peers on 8 metrics. Polarity-adjusted gap to median. RM rank 1–6. → gold.peer_benchmark
ANALYTICS Signal Detection + Priority Scoring
Analytics · Signals
analytics/01_validate_signals.py
~2700 metric pairs × 3 lags → Pearson CORR → |corr|>0.6 threshold → business prior gate → gold.signal_relationships
Analytics · Scoring
analytics/02_compute_priority_inputs.py
Normalises severity, persistence, peer_gap, commercial_weight, evidence_count to 0–1 scale for formula
Analytics · Events
analytics/03_build_event_windows.py
Links event_annotations.csv to metric anomaly windows. Not yet wired to UI in MVP.
Analytics · Gold
gold/04_build_priority_board.py
Weighted formula: 0.30×sev + 0.25×persist + 0.20×peer + 0.15×commercial + 0.10×evidence → rank top 50 → gold.priority_board
Analytics · Gold
gold/03_build_monthly_brief_inputs.py
Reads all 5 Gold tables → aggregates top-3 priorities, top-5 anomalies, top-4 signals, health summary → 1 row per month
5 Gold Delta tables complete
Infra · Script
scripts/build_local_snapshots.py
Reads 5 Gold Delta tables → converts to pandas → writes gold_*.csv to data/gold_snapshots/ (enables local/demo mode)
↓ CSV files ready for backend to load on startup ↓
Path B — Backend API + Frontend App
Backend Entry Point uvicorn app.main:app --reload
Frontend Entry Point npm run dev → vite dev server :5176
Backend · Entry
backend/api/app/main.py
lifespan() checks for gold_snapshots/ CSVs. Snapshot mode → load pandas DFs into app.state.data. Databricks mode → connect SQL Warehouse. Registers all 6 routers.
Backend · Client
app/clients/databricks.py
DatabricksClient — wraps databricks-sql-connector. Max 5 connections, retry on timeout. Only used in Databricks live mode.
