# ClubOS Web App

This folder contains the primary frontend for ClubOS.

The frontend should implement the workflow defined in:

- `AGENTS.md`
- `docs/product/clubos_mvp_spec.md`
- `docs/product/clubos_screen_blueprint.md`

Priority Board must remain the landing experience.

## Install

Use Node `20.16.0` and npm `10.8.1`.

```bash
../../scripts/bootstrap.sh
```

Or manually:

```bash
npm install
npm run dev
```
